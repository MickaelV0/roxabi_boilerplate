## Versioning

Roxabi Boilerplate follows [Semantic Versioning](https://semver.org/). Each release is tagged as `vMAJOR.MINOR.PATCH`.

- **Minor** bumps for new features (`feat` commits)
- **Patch** bumps for fixes and other changes
- While pre-1.0, breaking changes bump minor
